Setup virtual environment:
    CD into "AI"
    python -m venv .venv

To start virtual environment:
    CD into "AI"
    ./.venv/Scripts/Activate.ps1 (or .bat for regular terminal)

run a melee script, ie. example.py:
    Start virtual environment
    python3 example.py -e "Path to Slippi folder"
        (Find Slippi folder opening Slippi, Settings > Netplay > Open Config Folder)
